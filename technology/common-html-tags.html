Common html tags

HTML(Hypertext Markup Language) is a markup languge that is almost a must know for any web developer. It is used to 'markup' content as the name suggests. HTML tags tell the browser or whatever software used to browse what a certain element is, such as a table, paragraph, image and so on.

Basic html tags

<a>
Defines an hyperlink (commonly known as an anchor tag)
<abbr>
Defines an abbreviation or an acronym such as HTML
<address>
Defines contact information for the author of certain content
<article>
A semantic element for defining an article
<aside>
Defines content not part of an article but related to the article.
<audio>
Embeding audio/sound to the web
<b>
For making text bold
<br />
Used for inserting a line break
<button>
Defines a clickable item
<blockquote>
Defines a large block of quote
<canvas>
A container for graphics drawn using a programming language. Javascript is commonly used
<caption>
Defines a table caption
<code>
Defines a code statement. Monospace font is used for codes in HTML5.
<del>
Defines deleted text
<dl>
Defines a definition list
<dt>
Defines a definition term in a definition list
<div>
Used to divide block level content for common styling. It is commonly used in conjunction with CSS.
<dd>
Defines the definition of a term in a definition list
<em>
Emphasized text in italic(important)
<fieldset>
Groups related input elements in a form
<form>
Defines a form for taking user input
<footer>
Defines a document, article or section footer
<header>
Defines a document,article or section header
<h1> , <h2> , <h3> , <h4> , <h5> , <h6>
Defines headings from level one to level six respectively. Level one heading is the largest while level six is the smallest.
<hr />
Inserts an horizontal line.
<iframe>
Defines an inline frame
<i>
Defines Italicized text
<ins>
Defines Inserted text
<img>
Defines an image
<input>
Defines an input field in a form
<legend>
Defines a lebel for the fieldset element
<li>
Defines a list item in either ordered or unordered list
<mark>
Defines marked text with special styling
<nav>
A group of navigation links
<ol>
Defines an ordered list
<p>
Defines a paragraph
<pre>
Defines Pre-formatted text
<q>
<progress>
Defines the progress of a task
Defines a small line of quote
<section>
For dividing content in an article. (A section of an article)
<span>
Divides inline content for styling using CSS.
<sup>
Defines superscripted text
<svg>
Container for SVG graphics
<sub>
Defines subscripted text
<strong>
Defines an important text in bold.
<style>
Defines internal CSS styling
<script>
Defines client side scripts such as JavaScript code
<textarea>
Defines a multiline input element
<table>
Defines a table
<th>
Defines a table heading
<thead>
Defines a group of table headings
<tr>
Defines a table row
<td>
Defines table columns (table data)
<tbody>
Rows containing main table content
<tfoot>
Defines a table footer. Always at the bottom regardless of the placed position.
<video>
For embeding video to the web
