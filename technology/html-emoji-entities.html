Using Emojis in your website

If you are learning web development or are already a web developer, you might have one day thought about how to insert 'emojis' in your website. One solution can be to use HTML entities. They are supported by almost all modern browsers. There are very many HTML entities. We got the ones we think are the most interesting, colorful and also mostly used.
We have used Hexadecimal entities since we think they are the best to use. You don't have to remember all the entities.

To insert the entities using Hexadecimal values, start with &#x followed by the hexadecimal value then a semicolon ;
For example &#x2245;

Please, note that HTML entities are case sensitive. &#xF200; is not the same as &#xf200;

Please note, if you are using a laptop some of these entities might not work. Check out for safe entities.
