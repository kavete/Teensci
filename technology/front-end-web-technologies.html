Front End Web Technologies

Web development is divided into two main parts, Front End Web development and Back End Web development. Some might also include full stack development which is just a combination of front end and back end development. This article focuses on the front end technologies.

Front end web development majorly deals with programming the browser. What you see on a web page including the images, animations and functionality is majorly front end development. The main technologies used are HTML, CSS, Javascript, CSS frameworks and Javascript Libraries and Frameworks.

HTML (HyperText Markup Language)

HTML is one of the core technologies in web development. It is used to markup text (define how text will be displayed) in browsers (majorly). With introduction of HTML5 new capabilities are included such as embeding video, audio and many other features. XHTML is another alternative to HTML which is basically HTML conforming to XML (Extensible Markup Language) rules. The following is an example of a simple HTML code with a level one heading, an HTML entity and a paragraph.


<!doctype html>
 <html>
  <head>
  </head>
  <body>
   <h1>Heading</h1>

    <p>Hello there, my name is <b>Paragraph</b>.
    I am a paragraph of course &#x1F601;</p>
  </body>
 <html>
Output

Heading

Hello there, my name is Paragraph. I am a paragraph of course 😁 .

CSS (Cascading Style Sheets)

CSS is used to style HTML elements and to make a website responsive. CSS makes a website beautiful not just some plain text. Without CSS, websites could contain majorly plain text and maybe some images placed anyhow. With CSS, you can have color, animations and special behaviour for different devices (RWD -Responsive Web Design).

The following is a simple CSS code that changes the color of an element with id animation at given intervals.


<!doctype html>
<html>
 <head>
  <style>
   @keyframes colorchange {
     0% {background-color:blue;}
     100% {background-color:green;}
   } 

   #animation{

     animation-name:colorchange; 
     animation-duration:6s; 
     animation-iteration-count:infinite;
   }
  </style>
 <head>
 <body>
  <p id="animation" style="width:100px;height:100px;"> </p>

 </body>
</html>
Output

Javascript

Javascript is the programming language of the web. It is used to add interactivity to websites. With Javascript, you can include animations, change content dynamically, validate form data, add behaviour to elements and many more.

The following is a simple Javascript code used to alert the client



<!doctype html>
 <html>
  <head>
  </head>
 <body>
  <h1>Javascript example code<h1>
  
   <button onclick="hello()">Click me</button>

  function hello() {

    alert("This is Javascript in Action");

    }

 </script>
 </body>

CSS Frameworks/libraries

CSS libraries are just CSS files which make using CSS properties much easier. For example giving an element a certain class and automatically getting it styled in a certain way. There are many CSS frameworks incuding;

  1. ✔Bootstrap - Includes Javascript
  2. ✔Foundation
  3. ✔Tailwind
  4. ✔W3.css -Purely CSS
  5. ✔Bulma
  6. ✔Pure
  7. ✔Skeleton
  8. ✔Materialize

The most popular of the CSS libraries by far is Bootstrap, however if you are beginner I would recommend first checking on W3.css or the other mentiones libraries. CSS libraries also help in making your site responsive.You can even make your own CSS library to ease developing your web pages.

Javascript Libraries and Frameworks

Just like CSS libraries, Javascript libraries also ease the writing of Javascript code. Frameworks are a little bit more powerful, some even extending to Back end development. Most of the time pure Javascript (Vanilla Javascript) can involve lots of code for a task which can be done with a few lines when using a JS library. There are also many Javascript libraries and frameworks today incuding;

  1. ✔Jquery
  2. ✔ReactJs
  3. ✔AngularJS
  4. ✔Angular
  5. ✔VueJs
  6. ✔Polymer
  7. ✔Lodash

Jquery is very popular due to it's simplicity with power. Most people combine Jquery with one of the other mentioned libraries. There is no harm in learning all of them since each one of them has it's own strengths.

Other technologies

The above mentioned technologies are basically the main used, though their are others (not compulsory but important) such as git and bundlers such as webpack. Git version control makes you store all you code during all stages of development. You can switch from one version of your code to another.
